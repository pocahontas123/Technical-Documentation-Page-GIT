Basics of HTML5 ?

HTML and CSS are the ground of work of every websites. When you're consulting a site with your navigator, you have to know that, in the background, machineries are starting to make the websites able to show itself. The computer is basing on what HTML and CSS tell him to know what he have to show. HTML and CSS are two "languages" you have to speak to creat websites. It's the web navigators (firefox, chrome, edge...) which will traduct between the two languages and what you see on your screen. HTML (HyperText Markup Language): it's the skeleton of any websites. CSS (Cascading Style Sheets): it's the "look of" the websites. To work on HTML and css, you only need a simple windows text editor or a more advanced one like notepadd++ or sublime Text.

Tags and Attributs

In HTML we use what we call "Tags" and in css "Attributs". "Tags" are keywords surrounded by <> like <p>. It is use to tell to the navigator the nature of how you want the information between the tags, to be treated like. Generaly, Tags comes by group of two: an opening and one closing it: <title>This is a title</title>
<p>This is a paragraph</p>
You can find "self-closing tag" too. It mean that you don't need to use specific closing tag:
<img/>
About the "Attributs". They are a bit like the tags' options. They complet them to give more informations: They look commonly like this:
<tags attribut="value">
Here a real exemple:
<img src="localOrURLadress" alt="My Image description"/>

The basics structure of an HTML page

This code responde to a basic HTML5 page: 

								<!DOCTYPE html>
								<html>
									<head>
										<meta charset="utf-8" />
										<title>Titre</title>
									</head>

									<body></body>
								</html>

The first line is the DOCTYPE. It is essential and unique because it say that it is a HTML5 page.
The <html></html> represent the contents of the page.
The <head></head> represent general informations about the page.
The <body></body> represent the principal part of the page. It's where we write.
The <meta charset="utf-8" /> is the "encoding".
The <title></title> represent the title of the page:

The commentary

Commentaries are used to help you and others understanding your code better. It is only visible in the code itself:
<!-- this is a commentary -->
You have to understand that your code is completly visible by every one if it's on the web. In many browser, it's accessible by right clicking on the page then clic "Show the source code of the page".

The paragraphes

Most of the time, when we write simple text in a web page, we do it inside a paragraph. <p>This is a paragraph</p>
Like i said before, we write the the website content into the tag <body></body> If you want one way to "line break", you have to use more than one <p>: 

							<!DOCTYPE html>
							<html>
								<head>
									<meta charset="utf-8" />
									<title>Titre</title>
								</head>

								<body>
									<p>
										My Text 1
									</p>
									<p>
										My Text 2
									</p>
								</body>
							</html>

To do the same, you can use </br> where you want it to happen.

The Titles

To help you to manage your paragraphs better, you can use title. In HTML, we're lucky because we can directly use 6 tags for this use. It go from very big to very little:

<h2></h2>
<h3></h3>
<h4></h4>
<h5></h5>
<h6></h6>Very little boy

HighLight your text

All of these tags will have default 'css' associated to them. It mean that for exemple, one will be italic and the other one strong. But it's just the default 'look' and you should not use it like this. Their primary goal is not catch your eyes and the google robots eyes:
<em></em> it make the text italic

<strong></strong> it make the text strong

<strong></strong> it make the text strong

<mark></mark> it surround the text with a yellow color

It's important to point out that css is for the form and HTML the content.

Lists

Lists are commonly use to help to structurate our text and order informations. Two type of list exist:

  • Unordered list
  • Ordoned list

An unordered list look like this:

  • Tomatos
  • Carotts
<ul>
	<li>Tomatoes</li>
	<li>Carotts</li>
</ul>

An unordered list look like this:

  1. Tomatos
  2. Carotts
<ol>
	<li>Tomatoes</li>
	<li>Carotts</li>
</ol>

Create Links

Like you already know, the web is compose of multiple pages. How do you do come go from one to an other ? With the help of links ! To do this, you will need this tag:
<a href="http://url.com">Click me</a>
It can be use for directing some one to an webpage, a local page or to make him download a file.
<a href="http://url.com">Click me</a>
<a href="folder/myfile.html">Click me</a>
<a href="downloadme.zip">Click me</a>
And finaly you can create what is call "Anchor". It give you the possibility to go, in the same page or in an other one, somewhere in it by just clicking:
<h2 id="my-anchor">Title</h2>
Then where you want to go to:
<a href="#my_anchor">Click me</a>
